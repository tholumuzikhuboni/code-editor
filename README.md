## 🚀 About This Project

This project is a clean and intuitive online code editor that allows you to write and run HTML, CSS, and JavaScript directly in your browser. It's the perfect tool for quick experiments, learning web development fundamentals, or showcasing your front-end skills with live examples.

## ✨ Features

- ✅ **Tri-Panel Editor**: Write HTML, CSS, and JavaScript in separate, resizable editors.
- ✅ **Live Preview**: See your code come to life instantly on the same page.
- ✅ **Syntax Highlighting**: Enjoy beautiful and readable code powered by **CodeMirror**.
- ✅ **Developer Tools**: Includes options to format code, save snippets to local storage, and load them back in.
- ✅ **Responsive Design**: Fully optimized for a seamless experience on both desktop and mobile devices.

## 🚀 Getting Started

To set up and run this project locally, follow these simple steps:

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/tholumuzikhuboni/code-editor.git](https://github.com/tholumuzikhuboni/code-editor.git)
    ```

2.  **Navigate to the project directory:**
    ```bash
    cd code-editor
    ```

3.  **Open the `index.html` file in your browser:**
    You can simply double-click the file, or for the best experience, use a live server extension in your code editor (like VS Code's "Live Server").
